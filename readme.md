## GQLI

NOTE: this is still a work in progress

## Installation 

```
npm install -g gqli
```

## Create a project
```
gqli create [options] <project_name> [project_path]
```
Example:
```
gqli create ./first-project
```

## Available commands

Run help to view a list of available options and commands
```
gqli -h 
gqli -help
gqli --help
```
